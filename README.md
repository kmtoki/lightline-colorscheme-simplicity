# simplicity, lightline colorscheme
for terminal and gray backround color

## install
`cp simplicity.vim ~/<lightline path>/autoload/lightline/colorscheme/`

and add .vimrc

```vim:.vimrc
let g:lightline = {
\ 'colorscheme': 'simplicity',
\ 'separator': { 'left': '', 'right': '' },
\ 'subseparator': { 'left': '', 'right': '' }
\ }
```
